### Added

- 增加半开源许可文本、使用须知和 Release 模板
- 增加 README 截图展示、安装说明、使用说明和免责说明

### Changed

## [1.0.0] - 2026-03-17

### Added

- 初始公开版本
- 内嵌浏览器与多标签页工作台
- 媒体资源嗅探能力
- 资源预览、复制链接、单个下载和批量操作
- 音视频合并下载流程
- 收藏夹与平台导航
- 本地素材管理页面
- 基于 Electron、React、TypeScript 和 SQLite 的桌面应用架构
